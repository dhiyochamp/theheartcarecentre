üîå API Architecture

RESTful API endpoints with authentication

Sample API Routes REST API 
// Authentication
POST   /api/auth/register        - Register new user
POST   /api/auth/login           - User login
POST   /api/auth/logout          - User logout
POST   /api/auth/forgot-password - Reset password

// Departments
GET    /api/departments          - List all departments
GET    /api/departments/:slug    - Get department by slug
POST   /api/departments          - Create department (Admin)
PUT    /api/departments/:id      - Update department (Admin)
DELETE /api/departments/:id      - Delete department (Admin)

// Appointments
GET    /api/appointments         - List appointments
POST   /api/appointments         - Book appointment
PUT    /api/appointments/:id     - Update appointment
DELETE /api/appointments/:id     - Cancel appointment

// Doctors
GET    /api/doctors              - List all doctors
GET    /api/doctors/:id          - Get doctor details
GET    /api/doctors/:id/slots    - Get available slots

// Blog & News
GET    /api/blog                 - List blog posts
GET    /api/blog/:slug           - Get blog post
GET    /api/news                 - List news items
GET    /api/news/:slug           - Get news item
